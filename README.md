# my-norif-2025

## 概要

このプロジェクトは、タスクトレイに常駐するWindowsアプリケーションで、TCPで受信したメッセージをトースト通知として表示します。ローカルネットワーク内での利用を想定しています。

## ポート番号

ポート番号は第1引数または環境変数 `MY_NOTIF_PORT` で変更できます。デフォルトは `45654` です。

```powershell
# 第1引数で指定
> my-notif-2025.exe 12345
# 環境変数で指定
> $env:MY_NOTIF_PORT = 12345
> my-notif-2025.exe
```
